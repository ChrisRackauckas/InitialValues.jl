    UniversalIdentity.@def — Macro.
    UniversalIdentity.@def op [y = :x]

    Define a generic (left) identity for op.

    UniversalIdentity.@def op is expanded to

    op(::SpecificIdentity{typeof(op)}, x) = x
UniversalIdentity.hasidentity(::Type{typeof(op)}) = true

    For operations like push!, it is useful to define the returned value to be different from x. This can be done by using the second argument to the maco; i.e., UniversalIdentity.@def op [x] is expanded to

    push!(::SpecificIdentity{typeof(push!)}, x) = [x]
UniversalIdentity.hasidentity(::Type{typeof(push!)}) = true

    Note that the second argument to op is always x.
