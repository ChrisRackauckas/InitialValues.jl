    Initials.@def — Macro.
    Initials.@def op [y = :x]

    Define a generic (left) identity for op.

    Initials.@def op is expanded to

    op(::SpecificInitial{typeof(op)}, x) = x
Initials.hasinitial(::Type{typeof(op)}) = true

    For operations like push!, it is useful to define the returned value to be different from x. This can be done by using the second argument to the maco; i.e., Initials.@def op [x] is expanded to

    push!(::SpecificInitial{typeof(push!)}, x) = [x]
Initials.hasinitial(::Type{typeof(push!)}) = true

    Note that the second argument to op is always x.
